# README

The 3D periodontitis project mainly consists of three parts: **3D tooth instance segmentation**, **alveolar bone semantic segmentation**, and **traditional vision processing and grading of the segmentation results**. For the 3D teeth, this project adopts two implementation methods, with similar general ideas. The most crucial part is finding the **centroid of each tooth**, allowing for the segmentation of individual tooth regions based on the centroid. Then, individual teeth are segmented, and finally, all individual teeth are aggregated to form a single nii.gz file containing the tooth instance segmentation data.
